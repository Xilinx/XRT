XRT_API_EXPORT arg xrt::xclbin::kernel::get_arg ( int32_t  index) const

get_arg() - Get kernel argument at index.

Returns
The xrt::xclbin::arg object at specified argument index.

The memory connections of an argument is the union of the connections for each compute unit for that particular argument. In other words, for each connection of the argument returned by get_arg() there is at least one compute unit that has that connection.

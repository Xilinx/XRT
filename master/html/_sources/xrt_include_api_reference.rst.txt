XRT C/C++ API reference (``include/xrt``)
=========================================

This section is a **header-oriented API reference** for the native XRT library. It is organized to mirror the public headers under ``$XILINX_XRT/include/xrt``. For narrative usage (buffers, kernels, graphs, queues), see :doc:`xrt_native_apis`. For the full Doxygen build, see :doc:`xrt_native.main`.

**Build** (C++17, link ``xrt_coreutil``):

.. code-block:: shell

   g++ -std=c++17 -I$XILINX_XRT/include -L$XILINX_XRT/lib -o host host.cpp -lxrt_coreutil -pthread

**Layout**

* **Stable-style public headers** — top-level ``xrt_*.h`` under ``xrt/`` (device, buffer, kernel, graph, AIE, hardware context, UUID).
* **Experimental** — ``xrt/experimental/*.h`` (APIs may change; use with care).
* **Implementation details** — ``xrt/detail/*.h`` (not part of the stable application API; included transitively by public headers).
* **Deprecated** — ``xrt/deprecated/*.h`` (legacy C shims; prefer native APIs where documented).

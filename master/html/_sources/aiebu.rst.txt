aiebu
=====

AIE Binary Utilities (aiebu) provides utilities and libraries to work with AIE ctrlcode. aiebu provides AIE ctrlcode assembler and disassembler for *aie2p*, *aie2ps*, and *aie4* AIE architectures. aiebu also defines the ctrlcode ELF format and ctrlcode ISA.

For more information on aiebu please visit `AIE Binary Utilities (aiebu) <https://github.com/Xilinx/aiebu>`_.

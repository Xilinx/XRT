Linux Sys FileSystem NodesВ¶

xocl and xclmgmt drivers expose several sysfs nodes under the pci device root node. The sysfs nodes are populated by platform drivers present in the respective drivers.

xoclВ¶

The xocl driver exposes various sections of the xclbin image including the xclbinuuid on sysfs. This makes it very convenient for tools (such as xrt-smi) to discover characteristics of the image currently loaded on the FPGA. The data layout of xclbin sections are defined in file xclbin.h which can be found under runtime/core/include directory. Platform drivers XDMA, ICAP, MB Scheduler, Mailbox, XMC, XVC, FeatureROM export their nodes on sysfs.

Similar to PCIe drivers zocl driver used in embedded platforms exposes various sections of the xclbin image including the xclbinuuid on sysfs. This makes it very convenient for tools (such as xrt-smi) to discover characteristics of the image currently loaded on the FPGA. The data layout of xclbin sections are defined in file xclbin.h which can be found under runtime/core/include directory.

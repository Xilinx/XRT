865 * xrtBOAllocFromXcl() - Undocumented allocation from an xclBufferHandle
866 *
867 * @dhdl: XRT device handle on which the buffer is residing.
868 * @xhdl: The xcl buffer handle to convert to an xrtBufferHandle
869 * Return: xrtBufferHandle which must be freed using `xrtBOFree`
870 *
871 * Note that argument xclBufferHandle must be wrapped as
